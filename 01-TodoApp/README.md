Creating a Todo App in React

Description:
A simple Todo application built using React. This app allows users to add, delete, and mark tasks as completed. 

Approach:
1. Set up a new React project using Create React App.
2. Create components for the Todo List, Todo Item, and Input Form.
3. Use state to manage the list of tasks and their completion status.
4. Implement functions to handle adding new tasks, deleting tasks, and toggling the completion status of tasks.
5. Style the application using CSS to enhance the user interface.   

Learning Outcomes:
- Understanding the basics of React components and state management.
- Learning how to handle user input and events in React.
- Gaining experience in structuring a React application and organizing components.
- Developing skills in styling React applications using CSS.
- Enhancing problem-solving skills by implementing CRUD operations in a React application.

Technologies Used:
- React
- JavaScript
